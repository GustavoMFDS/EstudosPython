#  Estudos em Python

Bem-vindo ao meu repositório de **Estudos em Python**!  
Aqui você vai encontrar exercícios, projetos e experimentos que refletem minha jornada de aprendizado na linguagem Python.  
A ideia é manter tudo organizado, mostrar minha evolução e compartilhar conhecimento com quem também está estudando.

-------------------------------------------------------------------------------------------------------------------------------

##  Objetivo do Repositório
- Consolidar conceitos fundamentais de programação em Python.  
- Explorar diferentes áreas da linguagem: automação, análise de dados, APIs, interfaces gráficas e jogos.  
- Criar um portfólio prático que mostra minha evolução como desenvolvedor.  
- Servir como referência para revisões futuras e para outros estudantes.  

-------------------------------------------------------------------------------------------------------------------------------

## Estrutura do Repositório

O repositório está dividido em pastas temáticas, cada uma com um foco específico:

 Pasta              Conteúdo                                                                 
 `Automações`       Scripts para automatizar tarefas do dia a dia.                          
 `CodeCamp`         Desafios e exercícios resolvidos de plataformas de aprendizado.         
 `Exercicios`       Exercícios diversos para fixação de lógica e sintaxe.                   
 `Financas`         Scripts voltados para cálculos financeiros e organização de dados.      
 `Fixação`          Atividades voltadas para reforço de conceitos fundamentais.             
 `Pandas`           Estudos com a biblioteca Pandas para manipulação de dados.              
 `ProjetoFastAPI`   Projeto prático utilizando FastAPI para criação de APIs.                
 `RPG`              Protótipo de jogo em texto para aplicar lógica e estrutura de classes.  
 `Tkinter`          Interfaces gráficas simples usando a biblioteca Tkinter.                

---

##  Tecnologias Utilizadas
- **Python 3.11+**  
- **Bibliotecas padrão** (os, sys, datetime, etc.)  
- **Pandas** para análise e manipulação de dados  
- **FastAPI** para criação de APIs  
- **Tkinter** para interfaces gráficas
